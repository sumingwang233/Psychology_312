# 心理学312考研知识库

## 简介

这是一个公开的笔记仓库，将记录我27年心理学312考研的相关内容，整理时使用了Hermes agent。

本知识库用于系统化整理心理学312考研知识点，支持 Obsidian 直接打开和维护。

该知识库结构借鉴了[datawhale的大佬教程](https://datawhaler.feishu.cn/wiki/YesRwdINviiqPbkRkHlcGwuqnWe)

## 目录结构

```
心理学考研知识/
├── raw/               # 原始资料（永不编辑）
│   ├── articles/      # 公众号/网络文章
│   ├── papers/        # 学术论文
│   ├── transcripts/   # 课程转录稿
│   └── assets/        # 图片/附件
├── entities/          # 实体：人物/理论/学派
├── concepts/          # 核心概念定义
├── comparisons/       # 易混淆概念对比
├── queries/           # 问答/真题/考点
├── _templates/        # 页面模板
├── SCHEMA.md          # 规范定义
├── index.md           # 总索引
├── log.md             # 更新日志
└── README.md          # 本文件
```

## 快速开始

1. 用 Obsidian 打开本目录
2. 阅读 `SCHEMA.md` 了解规范
3. 从 `index.md` 导航到对应分区
4. 使用 `_templates/` 下的模板创建新页面
