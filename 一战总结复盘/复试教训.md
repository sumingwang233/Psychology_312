# 复试教训：华南师大与北京体育大学

> 一战进过两次复试但都未录取。每条教训背后都是真实踩过的坑。

---

## 一、华南师范大学脑科学与康复医学研究院

### 教训1：提前联系导师 = 提前开香槟

你说得很直白——"在没有绝对把握的情况下不要提前联系导师"。这件事的逻辑其实很简单：

**提前联系导师的唯一收益是"让导师提前认识你"，但代价是暴露你的位置。** 如果最终没进面，你在导师这边的第一印象就是"一个没进来的人"；如果进了但被刷，甚至可能让导师在面试时对你有额外判断。只有"确定能进面而且有硬实力"的人，提前联系才是一个正期望操作。

### 教训2：面试的"真诚陷阱"

你总结得也很直接——"自己就是一个演员"。这句话的底层逻辑是：

**复试面试不是"展示真实的你"，而是"展示符合这个学校/这个导师需求的那个你"。** 考官问的不是"你是个什么样的人"，而是"你是不是我们要的人"。所以：
- 不要在自己没准备过的问题上硬聊
- 不要把考研动机讲得太感性（"因为我热爱心理学"）
- 要把每个回答都指向一个信号：**我能在这里做出成果**

### 教训3：初试后的知识退化

"估分之后如果觉得自己能进面，就立刻进入复试的准备状态"——这条看似是时间管理，其实是**激活扩散模型**的应用：知识一个月不调取，提取强度会断崖式下降。复试考的是口头输出速度和实验设计能力，跟初试的书面答题是两套认知系统，需要重新激活。

## 二、北京体育大学心理学院

### 教训：双盲面试 ≠ 通用面试

一些学校采用双盲面试：考官不知道你的名字和背景，你也不能提。这种情况下，**不需要自我介绍，直接就是专业知识考察和实验设计。**

你的问题在于用华南的准备框架去应付北体——准备了一堆自我介绍的素材，结果发现根本用不上。

**正确的做法**：确定了调剂或第二志愿之后，立刻去研究该校的复试细则——往年真题、面试流程、是否双盲、实验设计占比。每个学校面试风格差异可以很大。

## 三、复试的总原则清单

| 阶段 | 原则 |
|------|------|
| 联系导师 | 除非有硬关系或确定进面且有突出优势，否则不提前联系 |
| 自我介绍 | 按1min/3min两个版本准备，但这个版本只有在"非双盲"时才用 |
| 专业问答 | 每个回答的逻辑链：问题是什么 → 理论/研究支持 → 实际应用/自己的理解 |
| 实验设计 | 这是复试的高频杀手题，提前准备4-5个经典研究方案的熟练口头输出 |
| 知识激活 | 初试后不超过两周开始复试准备，保持每天的知识提取节奏 |
