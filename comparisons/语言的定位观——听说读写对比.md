## 🧠 一、总体语言网络核心区域

| 脑区        | 位置         | 主要功能    |
| --------- | ---------- | ------- |
| **布洛卡区**  | 左侧额下回后部    | 运动性语言中枢 |
| **韦尔尼克区** | 左侧颞上回后部    | 听觉性语言中枢 |
| **角回**    | 顶叶-颞叶-枕叶交界 | 视觉性语言中枢 |
| **艾克纳尔区** | 额中回后部      | 书写性语言中枢 |

## 📊 二、听说读写的脑区定位与损伤症状对比表

| 语言技能        | 主要脑区                       | 功能作用                            | 受损后典型症状（失语症类型）     | 具体表现                                                                        |
| ----------- | -------------------------- | ------------------------------- | ------------------ | --------------------------------------------------------------------------- |
| **听**（语言理解） | **韦尔尼克区**<br>（左侧颞上回后部）<br> | 接收并理解口语语义<br>将声音转化为意义           | <br>**接收性失语/语言失认** | - 能流利说话，但内容空洞、错语多（如“把铅笔放到电话机里”）<br>- **听不懂他人说话**<br>- 无自知力（不觉得自己说错）         |
| **说**（语言表达） | **布洛卡区**<br>（左侧额下回）<br>    | 组织语法结构<br>规划发音动作                | **表达性失语/运动性失语**    | - **说话费力、电报式语言**（如“饭…吃…医院”）<br>- 语法缺失（无虚词、时态）<br>- **理解相对保留**<br>- 有自知力，常沮丧 |
| **读**（阅读）   | **角回**<br>（左侧顶-颞-枕交界）      | 将视觉文字转化为语音/语义<br>（“视觉→语音→语义”通路） | **语义性失语/听——视失语**   | - 视力正常，但无法识别文字<br>- 无法将口语和书面语正确匹配<br>- 关键词：***匹配 连接 跨通道***                  |
| **写**（书写）   | 艾克纳尔区                      | 将语义转化为书面符号<br>协调手部精细运动          | **失写症**            | - **能读，但不能写**<br>- 拼写错误、字形混乱<br>- 可能保留抄写能力（若运动通路完好）                         |
