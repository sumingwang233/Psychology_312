# 初试
1. 政治（**77**）
	1. 开始时间可以在8月之后，没必要太早，反而可能会有反作用
	2. 选择题注意时政的积累
	3. 多翻翻课本，一些细节可能会考（比如今年的北大）
	4. 大题要注意回答的逻辑，然后练练字，把版面分拿满
2. 英语（**80**）
	1. 需要精进翻译题，多积累词汇，今年就是有个词翻译错了才扣了分
	2. 作文要多练，今年作文分低于预期，个人感觉是自己写的时候太杂乱无章了，没有模板和结构
	3. 练字
3. 专业课312（**206**）
	1. 短板：实验设计、**测量**、计算题
	2. 这两年认知与神经科学的考察内容增加，需要补齐对应知识
	3. 教材上的经典实验设计需要不断去看
	4. 8月或9月后严格进行模拟，提高做题效率，312的题量很大，需要形成肌肉记忆和相应做题技巧
4. 学习习惯
	1. 每天深呼吸冥想10-20分钟，放空大脑
	2. 坚持锻炼身体
	3. 合理分配时间，早中晚要有不同的学习侧重
	4. 多画思维导图
	5. 事以密成，二战这一年要尽量断绝无关的社交网络
# 复试
1. 华南师范大学脑科学与康复医学研究院
	1. 在没有绝对把握的情况下不要提前联系导师，这属于提前开香槟，是很不好的行为
	2. 面试不要太"真诚"，要记住自己是来展示相匹配的能力和优势，以获得最大的上岸确定性为目的，自己就是一个演员
	3. 初试结束之后千万不能掉以轻心，估分之后如果觉得自己能进面，就立刻进入复试的准备状态，学习相应的知识，保持相应的知识激活强度
2. 北京体育大学心理学院
	1. 要看清楚学院复试的流程和细则，双盲面试不需要进行自我介绍，直接考察的就是专业知识和实验设计，不同学校的复试内容不同，具体问题具体分析
