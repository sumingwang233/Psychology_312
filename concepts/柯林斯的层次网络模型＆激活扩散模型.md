## 一、层次网络模型（Hierarchical Network Model）

### 核心思想  
知识以**分层树状结构**存储，概念按**类别层级**组织，**共享属性存储在最高可能的共同节点**，以节省认知资源（“认知经济性原则”）。
![[5bfb9b1821c90426025e8b033e4f8f81.png]]
### 结构特点
- **节点（Nodes）**：代表概念（如“金丝雀”、“鸟”、“动物”）
- **连线（Links）**：表示“是一种”（is-a）关系
- **属性（Properties）**：存储在最上层适用节点  
  例如：
  - “会唱歌” → 存于 **金丝雀**
  - “会飞” → 存于 **鸟**
  - “会呼吸” → 存于 **动物**

### 示例结构
```
        [动物]
         /  \
        /    \
     [鸟]    [鱼]
    /   \      \
[金丝雀][鸵鸟] [鲑鱼]
```
- “金丝雀会呼吸”？需从“金丝雀”→“鸟”→“动物”逐层检索“会呼吸”

### 实验证据：**句子验证任务**
- 被试判断句子真伪（如“金丝雀会飞？”、“金丝雀是动物？”）
- **预测**：层级距离越远，反应时越长  
  （“金丝雀是鸟” < “金丝雀是动物”）

### 模型局限
1. **典型性效应**（Typicality Effect）：  人的真实心理现象不一定和该模型相符
   - 人们判断“知更鸟是鸟”比“鸵鸟是鸟”更快，但两者层级距离相同。
2. **属性存储过于理想化**：  
   - 现实中“会飞”并非所有鸟共有（如鸵鸟），但人们仍将其与“鸟”强关联。

## 二、激活扩散模型（Spreading Activation Model）

### 核心思想  
对层次模型的改进：**放弃严格层级，采用语义相关性网络**。  
概念间通过**联想强度**连接，激活从一个节点**扩散**到相关节点，**扩散强度随距离和连接强度衰减**。
![[c9c8d5b7adf17be0303c9155c140919b.png]]
### 结构特点
- **非层级网络**：任意两个相关概念可直接连接
- **连线权重**：反映概念间**语义距离**或**关联强度**  
  （如“医生–护士”连线强于“医生–面包”）
- **激活扩散**：  
  当一个概念被激活（如“医生”），激活能量沿连线扩散，使相关概念（“医院”、“护士”、“听诊器”）更易被提取

### 解释现象
1. **语义启动效应**（Priming Effect）：  
   - 先呈现“医生”，再判断“护士”是否为词，反应更快——因“护士”已被激活。
2. **典型性效应**：  
   - “知更鸟”与“鸟”连接更强，激活更快。

### 优势＆发展

| 问题    | 层次模型 | 激活扩散模型       |
| ----- | ---- | ------------ |
| 典型性效应 | 无法解释 | ✅ 通过连接强度解释   |
| 非层级关系 | 无法处理 | ✅ 支持任意关联     |
| 否定判断  | 预测错误 | 更灵活（激活弱则反应慢） |

## 三、两模型对比总结

| 特征       | 层次网络模型    | 激活扩散模型        |
| -------- | --------- | ------------- |
| **结构**   | 严格树状层级    | 松散语义网络        |
| **连接**   | 仅“is-a”关系 | 任意语义关联        |
| **属性存储** | 最高层共享     | 分布式、可重复       |
| **信息提取** | 自上而下搜索    | 激活扩散          |
| **核心原则** | 认知经济性     | 语义相关性         |
| **解释力**  | 有限（仅层级知识） | 广泛（启动、联想、典型性） |

## 五、经典实验回顾

### Quillian (1969) 的验证实验
- **任务**：判断“鲨鱼是鱼？” vs “鲨鱼是动物？”
- **结果**：前者反应时更短 → 支持层级搜索
- **问题**：未控制典型性（“鲨鱼”对“鱼”的典型性高）

### Meyer & Schvaneveldt (1971) 的启动实验
- **任务**：判断“DOCTOR–NURSE”是否均为词
- **结果**：相关词对反应快于无关词对（如“BREAD–NURSE”）
- **支持**：激活扩散模型

---

## 总结

- **层次网络模型**：强调**结构经济性**，但过于 rigid。  
- **激活扩散模型**：强调**语义关联的动态性**，更贴近人类联想思维。  

> 💡 **关键洞见**：人类知识并非严格分类，而是由**强度各异的联想网络**构成——这正是激活扩散模型超越层次模型的核心所在。
