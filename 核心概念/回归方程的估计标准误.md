# 回归方程的估计标准误

## 定义

> 回归方程的**估计标准误**（Standard Error of Estimate，$S_{YX}$）是衡量回归方程预测精度的指标，表示因变量的实际值与回归方程预测值之间偏差的典型大小。

## 核心要点

- 公式：$S_{YX} = S_Y \sqrt{1 - r^2}$
- 当 $r \to 0$ 时，$S_{YX} \to S_Y$（**变大**，预测误差接近原始数据的离散程度）
- 当 $|r| \to 1$ 时，$S_{YX} \to 0$（**变小**，预测几乎完美）
- 这是2010年第58题的考察核心：**相关系数趋近于零时，估计标准误会提高**

## 公式推导与直观理解

### 代数推导

$$S_{YX} = \sqrt{\frac{\sum(Y - \hat{Y})^2}{N-2}} = S_Y\sqrt{1 - r^2}$$

其中 $r$ 是皮尔逊相关系数。

### 直观理解

```
S_Y = 原始数据的离散程度（没有X的时候Y有多散）
r²  = X解释了多少Y的变异
1-r² = X还剩下多少没解释的变异
S_Y√(1-r²) = 扣掉X能解释的之后，剩下的典型误差
```

> 🌰 想象预测一个班的考试成绩：
> - 如果完全不看任何预测变量（$r=0$），预测误差就是成绩本身的标准差——只能猜平均分
> - 如果知道学生 IQ（$r=0.5$），预测误差下降一些
> - 如果知道学生 IQ（$r=0.9$），预测误差大幅下降

## 2010年第58题解析

**题目**：在回归方程中，其他条件不变，X与Y相关系数趋近于零时，估计的标准误将会？

**解题**：
$$S_{YX} = S_Y \sqrt{1 - r^2}$$

当 $r \to 0$ 时：
$$\sqrt{1 - r^2} \to \sqrt{1 - 0} \to 1$$

$$S_{YX} \to S_Y \times 1 = S_Y$$

> ✅ **答案：提高**（B）。当相关系数趋近于零，预测变量对Y几乎无解释力，$S_{YX}$ 趋近于原始标准差 $S_Y$，比原来（有较强相关时）更大。

### 变化趋势图

```
r = 1.0  →  S_Y×0       = 0        （完美预测）
r = 0.8  →  S_Y×0.6     = 0.6S_Y   （较好预测）
r = 0.5  →  S_Y×0.87    = 0.87S_Y  （中等预测）
r = 0.1  →  S_Y×0.995   = 0.995S_Y （几乎无预测力）
r = 0    →  S_Y×1       = S_Y     （完全无预测力）
```

**结论**：$r$ 越大，$S_{YX}$ 越小——预测越准。

## 与其他统计量的关系

| 统计量 | 公式 | 含义 |
|--------|------|------|
| 决定系数 $R^2$ | $r^2$ | X解释了Y的变异比例 |
| 回归标准误 $S_{YX}$ | $S_Y\sqrt{1-r^2}$ | 预测的平均误差 |
| 皮尔逊相关系数 $r$ | $\frac{\text{Cov}}{\sigma_X\sigma_Y}$ | 线性相关的强度和方向 |
