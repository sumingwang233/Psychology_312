# 唐德斯减法反应时

## 定义

> 唐德斯（F.C.Donders, 1868）提出的用**减法逻辑**测量心理加工各阶段时间的方法。核心思路：通过比较不同反应条件下的反应时差异，分离出特定认知加工过程所需的时间。

## 核心要点

- 三种基本反应类型，按加工复杂度递增：A反应时（简单）→ B反应时 → C反应时
- 减法逻辑：$C - A$ = 辨别加工时间，$B - C$ = 选择加工时间
- 这是心理学史上第一个**客观测量心理过程时间**的方法

## 三种反应时类型

| 类型 | 名称 | 刺激数 | 反应数 | 加工内容 | 类比 |
|------|------|:---:|:---:|---------|------|
| **A反应时** | 简单反应时 | 1个 | 1个 | 感觉→运动 | 看到灯亮就按按钮 |
| **B反应时** | 选择反应时 | 多个 | 多个 | 感觉→辨别→选择→运动 | 红灯按左键，绿灯按右键 |
| **C反应时** | 辨别反应时 | 多个 | 1个 | 感觉→辨别→运动 | 只有红灯亮才按，绿灯不按 |

### 减法逻辑

```
A反应时 = 感觉加工 + 运动加工
B反应时 = 感觉加工 + 辨别加工 + 选择加工 + 运动加工
C反应时 = 感觉加工 + 辨别加工 + 运动加工

C - A = 辨别加工时间
B - C = 选择加工时间
```

> 💡 **记忆口诀**：A→案（简单），B→变（选择），C→扯（剥离出辨别）

## 前提假设

1. **加工过程是串行的**：一个阶段完成后再开始下一个
2. **各阶段独立**：一个阶段的时间不受其他阶段影响
3. **加工过程可叠加**：复杂反应时 = 各阶段简单反应时之和

> ⚠️ 这些假设在现代认知心理学中受到了挑战。但作为反应时研究的**起点**意义重大。
